# Dotfiles

This is a dotfiles directory created using stow

## Install using
- https://formulae.brew.sh/formula/stow (macOS)

## Setup

1. Clone the repository
```
$ mkdir ~/dotfiles
$ git clone https://github.com/axis1410/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
```
2. Use `stow` to create symlinks

```
$ stow .
```
